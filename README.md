# PathWise Smart Navigation System

## Overview
PathWise is a smart assistive project designed to help visually impaired individuals navigate safely using sensors and voice guidance.

## Features
- Obstacle detection
- Water detection
- Voice alerts
- Buzzer alerts
- Arduino implementation

## Technologies Used
- Arduino
- Sensors
- Embedded Systems
- C/C++
- IoT Concepts

## Project Goal
To improve accessibility and independent navigation using affordable technology.
